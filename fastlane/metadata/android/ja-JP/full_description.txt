WordPress for Android では、いつでもどこでもモバイルで Web 公開できます。 便利な追加機能を満載した Web サイト作成プログラムです。

作成
- すばらしいアイデアを配置するために Web 上に定位置を設けましょう。 WordPress for Android は Web サイトビルダーとブログメーカーの機能を兼ね備えたソフトウェアです。
- 豊富な選択肢のある WordPress テーマから最適な外観を選択し、写真、色、フォントをカスタマイズできるため、非常に個性的なサイトになります。
- 内蔵のクイックスタートヒントを参考にして、基本のセットアップを済ませ、新しいサイトを成功させましょう。

公開
- エディターを使えば、最新情報、ストーリー、写真付きエッセイの通知など、何でも作成できます。
- お使いのカメラやアルバムから写真や動画を埋め込むと、より生き生きとした説得力のある投稿やページになります。また、アプリ内蔵のプロ写真家の無料写真コレクションからも最適な画像を選んで活用できます。
- アイデアを下書き状態で保存しておいて都合のよいときに仕上げたり、将来の新規投稿のスケジュールを設定しておいて常に新しく魅力的なサイトを維持したりできます。
- 新規読者が投稿を見つけやすくなるようにタグやカテゴリーを追加して、訪問者数の増加を見守ります。

統計情報
- サイトの統計情報をリアルタイムでチェックし、サイト上のアクティビティを常に追跡します。
- 継続的に日ごと、週ごと、月ごと、年ごとの統計概要を調べ、最もトラフィックの大きい投稿やページを追跡します。

通知
- コメント、「いいね !」、新規フォロワーに関する通知が届くため、サイトに対する反応をリアルタイムで把握できます。
- 新しいコメントが追加されたらすかさず返信することで、スムーズな会話の流れを維持し、読者を大切にする姿勢を示すことができます。

Reader
- 数千のタグ別トピックを調べ、新規の投稿者や組織を見つけ、興味関心の対象をフォローします。
- 「保存」機能付きで後から編集できるため、投稿にじっくり取り組めます。

共有する
- 自動共有を設定しておくと、新しい投稿を公開したとき、ソーシャルメディアのフォロワーに通知できます。
- ソーシャル共有ボタンを追加できるため、訪問者が各自のネットワークでコンテンツを共有でき、ファンをアンバサダーに格上げできます。

WordPress の利点

ブログサービス、Web サイトビルダー、ソーシャルネットワークは市場にあふれています。 WordPress でサイトを作成するメリットは何でしょうか ?

WordPress は Web の1/3で使われています。 趣味のブログ、あらゆる規模の企業、オンラインストア、インターネット最大級のニュースサイトでも活用されています。 お好きなサイトの多くが WordPress で稼動している可能性もあります。

WordPress では、お客様がコンテンツの所有権を留保します。 ほかのソーシャルネットワークは、利用者を財貨とみなし、利用者の投稿したコンテンツの所有権を要求します。 その点、WordPress では、あらゆる公開コンテンツの権限が投稿者に帰属し、ほかのあらゆる場所で同時活用できます。

サイトを作成する Web サイトビルダーが必要な場合も、シンプルなブログメーカーが必要な場合も、WordPress は役に立ちます。 見事な設計、パワフルな機能、何でも開発できる自由があります。
